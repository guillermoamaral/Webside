# Retrieve usual categories

Retrieve popular categories used throughout the system.

**URL**: `/usual-categories`

**Method**: `GET`

## Success Responses

**Code** : `200 OK`

**Content**: `[string]`

**Example:**: `GET /usual-categories`.

```json
[
    "accessing",
    "actions",
    "construction",
    "event handlers",
    "initialization",
    "inquiries",
    "printing",
    "private",
    "services",
    "testing",
    "updating",
    "validation"
]
```
